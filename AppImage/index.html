Geeqie
A lightweight image viewer
Geeqie is a lightweight image viewer for Linux, BSDs and compatibles.

It can be used as a simple, fast, database-free image viewer,
but equally it can be used to manage large collections of images.

These AppImages have been created with Ubuntu 20.04.
